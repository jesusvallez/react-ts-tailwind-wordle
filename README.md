## Getting Started

To get a local copy up and running follow these simple example steps.

### Installation

1. Clone the repo
   ```sh
   git clone git@github.com:jesusvallez/react-ts-tailwind-wordle.git
   ```
2. Install NPM packages with Yarn
   ```sh
   npm i
   ```
3. To start web
   ```js
   npm run dev
   ```
